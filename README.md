# gzqh-portable

Portable, sanitized `gzqh` failover package.

## One-line install

```bash
curl -fsSL https://raw.githubusercontent.com/saotu/gzqh-portable/main/install-online.sh | bash
```

## Offline install

```bash
unzip gzqh-portable.zip
cd gzqh-portable
bash install.sh
gzqh
```

## Uninstall

```bash
bash uninstall.sh
```
